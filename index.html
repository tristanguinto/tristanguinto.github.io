RADBLOCK - Senior Design Project

For my senior design project at UCI, my team and I designed a radio that would detect advertisement through the current audio and switch to another channel automatically. We used a Raspberry Pi3 in conjunction with TensorFlow in order to use the machine learning model on a portable device. I was mainly responsible for the hardware aspect of this project, which dealt with primarily C language programming and socket integration with our Python TensorFlow model. This project ended up getting the Dean's Choice Award at the 2019 Winter Design Review.
